Паттерн Итератор (Iterator) — это поведенческий паттерн проектирования, который предоставляет способ последовательного доступа к элементам коллекции без раскрытия её внутренней структуры. Этот паттерн позволяет клиенту перебирать элементы коллекции, не зная деталей её реализации.

Структура паттерна Итератор:
Интерфейс итератора определяет методы для перемещения по коллекции и получения текущих элементов.
Конкретный итератор реализует интерфейс итератора и держит текущее положение в коллекции.
Агрегат (или коллекция) предоставляет метод для создания итератора.

Преимущества использования паттерна Итератор:
Скрывает внутреннюю структуру коллекции от клиента.
Позволяет одновременно использовать несколько итераторов для одной коллекции.
Упрощает навигацию по коллекциям сложных структур данных.
